1) входная дверь
	_entrance_door
		web_server commands:
			open  - открыть
			close - закрыть
		dev states: opened/closed

2) таймер
	_timer
		web_server commands:
			activate/:value - открыть
			stop            - остановить
		dev events:
			ready - сработал
		dev states: active/idle/ready
		parameters:
			current_value  - текущее значение таймера
			value      - время, отсчитываемое таймером

4) кнопка шкафа

5) дверь шкафа
	_locker_door
		web_server commands:

		dev states: opened/closed

6) подставка многогранника
